# Preface {#Contributor_Preface}

## Overview of NGIpkgs {#Contributor_Overview_of_NGIpkgs}

The NGIpkgs repository packages into Nix and provides NixOS services
for softwares that are funded by the [Next Generation Internet](https://www.ngi.eu) (NGI) initiative
of the [European Commission](https://commission.europa.eu)
through the [NLnet Foundation](https://nlnet.nl/).

## How to use NGIpkgs' Contributor Manual {#Contributor_How_to_use_NGIpkgs_Contributor_Manual}

If you would like to:
- be guided with hands-on examples:
  - while learning a craft with NGIpkgs, go to [Tutorials for contributors](#Contributor_Exercise_to).
  - while performing a craft on NGIpkgs, go to [Recipes for contributors](#Contributor_How_to).
- browse and ponder:
  - while learning a craft with NGIpkgs, go to [Explanations for contributors](#Contributor_Why_to).
  - while performing a craft on NGIpkgs, go to [Descriptions for contributors](#Contributor_What_is).
