# Preface {#User_Preface}

## Overview of NGIpkgs {#User_Overview_of_NGIpkgs}

The NGIpkgs repository packages into Nix and provides NixOS services
for softwares that are funded by the [Next Generation Internet](https://www.ngi.eu) (NGI) initiative
of the [European Commission](https://commission.europa.eu)
through the [NLnet Foundation](https://nlnet.nl/).

## How to use NGIpkgs' User Manual {#User_How_to_use_NGIpkgs_User_Manual}

If you would like to:
- be guided with hands-on examples:
  - while learning a craft with NGIpkgs, go to [Tutorials for users](#User_Exercise_to).
  - while performing a craft on NGIpkgs, go to [Recipes for users](#User_How_to).
- browse and ponder:
  - while learning a craft with NGIpkgs, go to [Explanations for users](#User_Why_to).
  - while performing a craft on NGIpkgs, go to [Descriptions for users](#User_What_is).
