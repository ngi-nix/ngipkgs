# How to install NGIpkgs using `cachix`?

For your convenience, you may enable NGIpkgs' public build cache,
either imperatively by using [cachix](https://www.cachix.org):
```bash
cachix use ngi
```

You can revert this trust with:
```bash
cachix remove ngi
```
