# How to learn NixOS?

{#Contributor_How_to_learn_NixOS_Resources}
## Resources
- Nixpkgs: https://nixos.org/manual/nixpkgs/unstable/
- CppNix: https://nix.dev/manual/nix/latest
- Lix: https://docs.lix.systems/manual/lix/stable/

{#Contributor_How_to_learn_NixOS_Sources}
## Sources

- Browse `nixpkgs` for examples with `git grep` or `nvim-telescope`, or `<you-name-it>`
  mostly in `nixos/modules` and `nixos/tests`.
- Query the whole Github for examples.
    Tip(usability/operability/speed): use a Web browser bookmark with `%s` in the URL, eg:
    - URL: `https://github.com/search?type=code&q=%s`
    - Keyword: `gh`
    then just type `gh <your-query>` in the URL bar.
