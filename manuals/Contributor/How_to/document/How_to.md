# How to document `How to` recipes?

See:
- [diataxis.fr/how-to-guides/](https://diataxis.fr/how-to-guides/)
- [diataxis.fr/tutorials-how-to/#tutorials-how-to](https://diataxis.fr/tutorials-how-to/#tutorials-how-to)

TL;DR:
- Assume the user knows what they want to achieve.
- Focus on tasks or problems, not machinery.
- No digression, explanation, teaching.
- Take a general approach.
- Describe different routes to the same destination.
- Action and only action.
