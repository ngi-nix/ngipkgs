# What is security tool `systemd-analyze security`?

[`systemd-analyze security`](https://www.freedesktop.org/software/systemd/man/latest/systemd-analyze.html#systemd-analyze%20security%20%5BUNIT...%5D)
is a tool bundled within `pkgs.systemd` that helps to review the settings
by printing a detailed report of the state of each option,
with ponderations summed into a global hardening level.
